## Adding New Posts

Create a new `.mdx` file with frontmatter:

```yaml
---
title: "Post Title"
date: "YYYY-MM-DD"
author: "Axat Bhardwaj"
description: "Brief description for SEO"
tags: ["tag1", "tag2"]
---
```
