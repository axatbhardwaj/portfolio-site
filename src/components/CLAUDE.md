# src/components/

React UI components for the portfolio site.

## Files

| File                       | What                                     | When to read                                                         |
| -------------------------- | ---------------------------------------- | -------------------------------------------------------------------- |
| `header.tsx`               | Site header with name/title              | Modifying header content, styling                                    |
| `navbar.tsx`               | Navigation bar with links                | Adding nav items, modifying navigation                               |
| `github-activity.tsx`      | GitHub contribution calendar (client)    | Modifying contribution display, theme colors, debugging data loading |
| `project-card.tsx`         | Individual project card                  | Modifying project card layout, styling                               |
| `project-showcase.tsx`     | Grid/carousel of project cards           | Modifying project grid layout, carousel behavior                     |
| `spotlight-showcase.tsx`   | Featured projects carousel               | Modifying spotlight section, carousel navigation                     |
| `detailed-project-card.tsx`| Expanded project card with achievements  | Modifying detailed project display                                   |
| `blog-section.tsx`         | Blog posts section on homepage           | Modifying homepage blog preview                                      |
| `posts.tsx`                | Blog posts list component                | Modifying posts list rendering                                       |
| `posts-list.tsx`           | Simple posts list wrapper                | Modifying posts list structure                                       |
| `post-item.tsx`            | Individual post list item                | Modifying post item layout                                           |
| `section-list.tsx`         | Generic section with item list (work)    | Adding new list sections, modifying work history                     |
| `links-section.tsx`        | Social links section                     | Adding/modifying social links                                        |
| `scramble-text.tsx`        | Animated text scramble effect            | Modifying text animations                                            |
